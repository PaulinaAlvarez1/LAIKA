# LAIKA
Prueba para mobile developer.

Esta prueba incluye:
1. Diseño de home screen
2. Manejo de redux para cambios de estado
3. Manejo de hooks para estados locales
4. Scripts para generación de apk

plus: 

- Splash screen
- Logos
- Fuentes

Instrucciones

1. yarn install o npm install
2. pod install
3. yarn android

Generación de apk

- Yarn build:android
